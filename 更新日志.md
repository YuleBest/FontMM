# FontMM 更新日志

## V1.3 Alpha：测试版本
- 新功能：搜索、下载并解包小米国际版主题的字体
- 新功能：xml 和 json/yaml 的互转
- 新功能：拆分 TTC 字体文件
- 新功能：现在刷入时会自动检查文件，并设置权限
- 修复：部分情况下字体不生效/消失的问题
- 修复：代码语法错误
> TTC 是一种字体集合格式，它将多个相关的 TTF 字体打包进同一个文件，现在你可以方便地对此类字体进行拆分

## V1.2：快捷更换字体，一键备份模块
- 新功能：使用 Action 直接更换字体，无需重新打包 (alpha)
- 新功能：备份当前已安装的模块
- 新功能：检查更新脚本 (beta)
- 新功能：反馈脚本，运行后截图给开发者，帮助改进项目 (alpha)

## V1.2 Beta：测试版本
- 新功能：添加检查更新脚本
- 优化：字体更换移动到 Action 二级目录

## V1.2 Alpha：测试版本
- 新功能：使用 Action 直接更换字体，无需重新打包
- 新功能：增加反馈脚本

## V1.1：打包更快，模块更小
- 由原来的全文件压缩改为追加压缩，大幅降低打包时间，现在最快只需 3 秒即可完成打包！
- 把复制 N 个文件流程放到了模块的刷入脚本，大幅减小模块的体积（小于 150 MB！）
