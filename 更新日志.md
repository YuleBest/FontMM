# FontMM 更新日志

## V1.2：快捷更换字体，一键备份模块
- 新功能：使用 Action 直接更换字体，无需重新打包 (alpha)
- 新功能：备份当前已安装的模块
- 新功能：检查更新脚本 (beta)
- 新功能：反馈脚本，运行后截图给开发者，帮助改进项目 (alpha)

## V1.2-beta：测试版本
- 新功能：添加检查更新脚本
- 优化：字体更换移动到 Action 二级目录

## V1.2-alpha：测试版本
- 新功能：使用 Action 直接更换字体，无需重新打包
- 新功能：增加反馈脚本

## V1.1：打包更快，模块更小
- 由原来的全文件压缩改为追加压缩，大幅降低打包时间，现在最快只需 3 秒即可完成打包！
- 把复制 N 个文件流程放到了模块的刷入脚本，大幅减小模块的体积（小于 150 MB！）
