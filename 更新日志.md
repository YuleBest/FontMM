# FontMM 更新日志

## V1.3
- 新功能：解析中文简体字体的名称作为模块的名称（可关闭）
- 新功能：搜索、下载并解包小米国际版主题的字体
- 新功能：xml 和 json/yaml 的互转
- 新功能：拆分 TTC 字体文件
- 新功能：现在刷入时会自动检查文件，并设置权限
- 重构：使用自制的 fonttool-rs 工具代替 fonttool（https://github.com/YuleBest/fonttool-rs）
- 优化：环境处理，确保权限正确
- 优化：加入 libbz2 动态库解决部分环境（如虚拟机）无法运行 zip
- 修复：部分情况下字体不生效/消失的问题
- 修复：一些环境下无法运行可执行程序的问题
- 修复：部分环境泄露问题 
- 修复：代码语法错误
- 修复：计时器取两位小数不生效的问题

## V1.3 Beta
- 新功能：尝试解析 `ch.ttf` 字体的元信息，使用字体名作为模块的 `name`（需在 `开始打包.sh` 文件内配置 `USE_FONT_META="1"`）
- 修复：部分环境泄露问题 (dff1ae9)
### V1.3 Beta.2

    - 修改代码适应此更改

## V1.1：打包更快，模块更小
- 由原来的全文件压缩改为追加压缩，大幅降低打包时间，现在最快只需 3 秒即可完成打包！
- 把复制 N 个文件流程放到了模块的刷入脚本，大幅减小模块的体积（小于 150 MB！）
